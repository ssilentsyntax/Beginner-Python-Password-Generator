# Password Generator

A simple Python script to generate secure passwords with customizable options for length, numbers, and special characters.

## Features

- Generate passwords of any minimum length.
- Option to include numbers.
- Option to include special characters.
- Simple command-line interface.

## Usage

1. Make sure you have Python installed (Python 3 recommended).
2. Download or clone this repository.

## Example 
1. Enter minimum password length: 'number'
2. Include numbers? (y/n): y
3. Include special characters? (y/n): y
4. Generated password: P@ssw0rd!A1b
